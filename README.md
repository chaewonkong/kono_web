## Installation

### Install json-server
```shell
npm install -g json-server
```

### Install Project
```shell
yarn install
```

### Start Project
In order to start project, need to start json-server first:
```shell
json-server db.json --port 3004
```
The port has to be set 3004 for CRA server runs on 3000.

Now run the project:
```shell
yarn start
```
